# 光伏组件-逆变器及线缆匹配查询工具

一个用于光伏发电系统中组件、逆变器、并网箱及线缆选型匹配的在线查询工具。

## 功能特性

- **区域选择**：支持全国各省、市、区县三级联动选择
- **容配比配置**：支持 1.0 倍、1.1 倍、1.2 倍容配比选择
- **组件系列**：支持 NEG21 (715-720W)、NEG21 (730-740W)、NEG22 (790-800W) 三个系列
- **智能匹配**：根据输入条件自动计算并输出：
  - 逆变器配置方案
  - 逆变器交流铜线规格
  - 并网箱配置方案
  - 并网箱交流铝线规格

## 技术规范

- 逆变器输出线必须使用铜线 (ZR-YJV-0.6/1kV)
- 交流电缆长度 ≤50m，线径参考表格标准
- 50m < 长度 ≤100m，线径比规定大一号
- 100m < 长度 ≤150m，线径比规定大二号
- 长度 >150m，需联系交付人员处理

## 使用说明

1. 选择省份、城市、区县
2. 选择容配比（自动锁定为 1.2 倍正常超配）
3. 选择组件系列
4. 输入组件数量（10-330 片）
5. 点击"智能匹配查询"获取结果
