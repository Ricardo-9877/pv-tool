# 湖南版730优化需求规格文档

## 一、需求概述

针对湖南省光伏系统组件匹配工具的专用配置优化，基于现有 NEG21_730 组件系列，添加湖南地区专属的逆变器挡位配置。

## 二、需求详情

### 2.1 湖南省组件系列限制

- **功能描述**：当用户选择「湖南省」时，组件系列下拉框只显示「730系列」选项，715系列和800系列不可选
- **实现位置**：`initCascade()` 或 `cascadeUpdate()` 中，根据省份选择动态控制组件系列选项

### 2.2 湖南专用逆变器配置表

在 `DB` 对象中新增 `HUNAN_730` 数据集，结构如下：

| 容配比 | 组件数量范围 | 逆变器配置 | 并网箱功率 |
|--------|-------------|------------|-----------|
| 1.2倍(正常) | 65~68 | 17+25 (42kW) | 50kW |
| 1.2倍(正常) | 69~72 | 20+25 (45kW) | 50kW |
| 1.2倍(正常) | 73~76 | 17+30 (47kW) | 50kW |
| 1.1倍 | 60~62 | 17+25 (42kW) | 50kW |
| 1.1倍 | 63~66 | 20+25 (45kW) | 50kW |
| 1.1倍 | 67~69 | 17+30 (47kW) | 50kW |
| 1倍 | 55~56 | 17+25 (42kW) | 50kW |
| 1倍 | 57~60 | 20+25 (45kW) | 50kW |
| 1倍 | 61~63 | 17+30 (47kW) | 50kW |

### 2.3 逆变器组成说明

| 逆变器功率 | 组成方式 | 并网箱功率 |
|-----------|---------|-----------|
| 42kW | 17kW + 25kW | 50kW |
| 45kW | 20kW + 25kW | 50kW |
| 47kW | 17kW + 30kW | 50kW |

### 2.4 其他数量段处理

- 湖南地区在上述数量范围之外的其他组件数量，继续使用现有的 NEG21_730 配置表
- 即：10~54块、77~320块 沿用现有配置

### 2.5 并网箱与线缆

- 新增的 42kW/45kW/47kW 逆变器对应的并网箱功率为 **50kW**
- 并网箱备注（b字段）保持现有湖南各地区的配置不变
- 线缆规格根据逆变器总功率查表确定

## 四、改造点清单

1. **组件系列联动逻辑修改**（`cascadeUpdate` 或新建函数）
   - 检测省份是否为「湖南省」
   - 若是，组件系列只保留「730系列」选项

2. **湖南专用查询逻辑**
   - 在 `queryMatch()` 中增加湖南地区判断
   - 优先查询 `HUNAN_DB`，未命中则回退到 `DB.NEG21_730`

3. **并网箱线缆逻辑适配**
   - 确保 42kW/45kW/47kW 按 50kW 功率查表

## 五、验收标准

1. 选择湖南省时，组件系列下拉框仅显示「730系列」
2. 选择其他省份时，组件系列下拉框显示「715系列」「730系列」「800系列」
3. 湖南地区输入特定数量（55~76）时，返回新增的 42kW/45kW/47kW 逆变器配置
4. 并网箱功率显示为 50kW，线缆规格正确
5. 湖南地区其他数量（55以下、76以上）使用现有 730 配置
